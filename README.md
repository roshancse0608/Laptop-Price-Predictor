# Laptop Price Predictor

## Overview

The Laptop Price Predictor is a machine learning project that estimates the price of a laptop based on its specifications, including brand, RAM size, storage capacity, processor generation, and screen size.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Google Colab
* GitHub

## Features

* Creates a laptop specifications dataset
* Preprocesses categorical data using one-hot encoding
* Trains a Linear Regression model
* Predicts laptop prices based on user input
* Visualizes actual versus predicted prices

## Dataset Attributes

* Brand
* RAM_GB
* Storage_GB
* Processor_Gen
* Screen_Size
* Price

## Machine Learning Algorithm

* Linear Regression

## Results

The model achieved a good R² score, demonstrating effective price prediction based on laptop specifications.
